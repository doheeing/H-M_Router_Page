<h2>구현기능</h2>

  <br>유저는 메뉴와 상품들을 볼 수 있다.</br>
  <br> 유저는 로그인을 할 수 있다.</br>
   <br>유저는 상품디테일을 보기 위해 로그인을 해야한다.</br>
   <br>로그인한 유저는 상품디테일정보를 볼 수 있다.</br>
   <br>유저는 상품을 검색할 수 있다.</br>
   <br>유저는 로그아웃할 수 있다.</br>
   <br>유저는 좋아하는 상품에 표시를 할 수 있다.</br>
